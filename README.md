🎨 Color Heaven | Professional Color Engine
Color Heaven is a sophisticated web-based color palette generator built for designers and developers who value precision. It bridges the gap between creative branding and technical implementation by providing mathematical color harmony and accessibility checks in one fluid interface.

🚀 Key Features
1. Advanced Color Theory Engine
The core logic generates 5 types of professional palettes using precise HSL (Hue, Saturation, Lightness) calculations:

Analogous: Adjacent colors for natural harmony.

Monochromatic: Variations in lightness/saturation of a single hue.

Triadic: Three balanced colors spaced evenly around the wheel.

Complementary: High-contrast opposite pairings.

Tetradic (Double-Complementary): Rich, complex four-color schemes.

2. WCAG Accessibility Checker
Every generated color is automatically tested against the WCAG 2.1 standards. The engine calculates the contrast ratio and provides real-time ratings:

✅ AAA: Highest accessibility.

✅ AA: Standard web accessibility.

⚠️ Fail: Low contrast (not recommended for text).

3. Developer-Centric Workflow
Click-to-Copy: Instant HEX code copying to clipboard.

CSS Export: One-click download of the entire palette as a ready-to-use :root CSS variables file.

🛠️ Technical Stack
HTML5: Semantic structure.

CSS3: Custom properties (variables), Grid, Flexbox, and staggered animations.

Vanilla JavaScript: pure ES6 logic for HEX/HSL conversion and DOM manipulation. Zero Dependencies.
